Portal SENNOVA APP

Esta plataforma fué programada por el equipo de desarrollo de Sennova centro de comercio del SENA del cual hice parte en el año 2017.

El aplicativo, permite la gestión y presentación gráfica de información referente a diferentes diagnósticos empresariales, así como la autenticación de usuarios e implementación de servicios de actualización de datos en base de datos haciendo uso de:

  • Patrón MVC de PHP
  • Uso de estilos propios CSS

GEOPORTAL SENNOVA

Esta plataforma (SPA) fué programada por el equipo de desarrollo de Sennova centro de comercio del SENA del cual hice parte en el año 2018.

El aplicativo, permite el diligenciamiento en línea de diferentes formularios y la aplicación de un modelo matemático para la presentación gráfica de información estadística, además se utilizan mapas dinámicos para osociar la información presentada a una región geográfica en especifico.

Tecnicamente el geoportal se programo utilizando:

  • Módulos y librerías JavaScript
  • Estilos css propios
  • Bootstrap 4

COTINGA

Esta aplicación nativa para android, permite ayudar en el proeso de avistamiento de aves en el municipio de jardín, Antioquia-Colombia.
Tecnicamente la aplicación permite al usuario:

  • Acceder a una base de datos local con información de mas de 100 aves diferentes
  • Tomar fotografías desde la app y compartirla con redes sociales
  • Acceder a información turística y de guías almacenados en una base de datos en línea

V Simposio SENNOVA

Esta aplicación nativa para android, brinda infromación de utilidad para los participantes del V simposio nacional SENNOVA.
Tecnicamente la aplicación permite al usuario:

  • Acceder a una base de datos local con información de todas las ponencias del evento
  • Aplicar diferentes filtros de búsqueda de ponencias en la agenda del evento
  • Obtener información de ayuda e interés general

Sennova GTH

Sennova GTH hace parte de 8 apps creadas para el SENA como apoyo a las actividades de diagnostico empresarial realizadas por aprendices del centro de comercio.
Tecnicamente la aplicación permite al usuario:

  • Autenticación de usuarios en línea
  • Escritura y actualización de base de datos en Línea
  • Diligenciamiento y sincronización de formularios en tiempo real

METRO SAI

METRO SAI, fué un proyecto desarrollado en 2016 como parte de mi pasantía en el metro de Medellín, la plicación nativa para android permitió el control remoto de las diferentes UPS del sistema en sus dos lineas comerciales.
Tecnicamente la aplicación permite al usuario:

  • Acceso remoto a las UPS utilizando el protocolo SNMP
  • Ubicación satelital de las diferentes UPS del sistema
  • Información técnica general de los sistemas de alimentación ininterumpida del metro
